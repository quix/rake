                      run
                      ()
                    

                    

                      

                      Run the Rake application.  The run method
                      performs the following three steps:
                      

                      
    
                      
  • 
                      Initialize the command line options (init).
                      
    
                      
    • 
                      
  • 
                      Define the tasks (load_rakefile).
                      
    
                      
    • 
                      
  • 
                      Run the top level tasks (run_tasks).
                      
    
                      
    • 
                      

                      

                      If you wish to build a custom rake command, you should call init
                      on your application.  Then define any tasks.  Finally, call top_level to run your
                      top level tasks.
